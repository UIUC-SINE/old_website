The naive method of extending phase-correlation for subpixel registration is to zeropad by a factor times the original image size before computing the IFFT to obtain a finer sampling grid. However, when we desire finer subpixel estimates (like ), it becomes intractible to compute and store such a large IFFT.

This is a test of the subpixel phase-correlation image registration method proposed by Guizar-Sicairos, Thurman, Fienup. The paper proposes that if we can roughly estimate the image offset, we need only compute a small region of the inverse fourier transform using IDFT instead of IFFT.
